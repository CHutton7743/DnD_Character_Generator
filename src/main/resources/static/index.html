5th Edition Character Forge

What is this tool?

A passion project of a dnd lover

As an experienced DM I always disliked how hard it can be for new players to get started and get into the game. Or how much effort it takes for a dm to make side characters. My goal with this project was to make it easy for all players to get started playing!

Random characters and NPC's

I am currently working on support to add the ability for the tool to create random characters and NPC's, this should help DM's or even people who want something unexpected to be able to get going faster.

What is the outcome?

This tool will help you to create characters and deliver you a character sheet for the created character in PDF format.
